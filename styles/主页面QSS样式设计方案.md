# 主页面 QSS 样式设计方案

## 1. 侧边栏导航按钮样式

### 1.1 选中状态按钮（QPushButton:checked）

**视觉特征分析：**
- 背景色：浅蓝色/淡紫色（类似 `#E8F0FE` 或 `#D6E4FF`）
- 圆角：明显圆角边框（约 8-12px）
- 图标：左侧 2x2 网格图标（仪表盘图标）
- 文字：深色文字"仪表盘"
- 布局：图标和文字水平排列，有适当间距

**推测的 QSS 样式：**

```css
/* 选中状态的按钮样式 */
QPushButton:checked {
    background-color: #E8F0FE;  /* 浅蓝色背景 */
    border: none;
    border-radius: 8px;
    padding: 8px 16px;
    color: #1a73e8;  /* 深蓝色文字 */
}

QPushButton:checked:hover {
    background-color: #D2E3FC;  /* 悬停时稍深 */
}
```

**关键属性说明：**
- **`checked` 伪类**：表示按钮处于选中/激活状态
- **背景色**：使用浅色背景突出选中项
- **无边框**：`border: none` 移除默认边框
- **圆角**：`border-radius` 提供现代感
- **内边距**：`padding` 提供舒适的点击区域

**适用场景：**
- 侧边栏导航菜单
- 用于指示当前激活的页面
- 常见于仪表盘、车辆管理等功能模块

### 1.2 鼠标悬浮状态按钮（QPushButton:hover）

**视觉特征分析：**
- 背景色：浅灰色/淡蓝色（类似 `#f1f3f4` 或 `#e8f0fe`）
- 圆角：明显圆角边框（约 8-12px）
- 图标：左侧用户图标（用户管理图标）
- 文字：深色文字"用户管理"
- 边框：无明显边框

**推测的 QSS 样式：**

```css
/* 鼠标悬浮状态按钮 */
QPushButton:hover {
    background-color: #f1f3f4;  /* 浅灰色背景 */
    border: none;
    border-radius: 8px;
    padding: 8px 16px;
    color: #202124;  /* 深色文字 */
}
```

**关键属性说明：**
- **`hover` 伪类**：表示鼠标悬浮在按钮上
- **背景色**：浅灰色提供视觉反馈
- **无边框**：保持简洁外观
- **圆角**：与选中状态保持一致

### 1.3 基础/普通状态按钮（QPushButton）

**视觉特征分析：**
- 背景色：白色或透明（无明显背景色）
- 圆角：无明显圆角或很小的圆角
- 图标：左侧摄像头图标
- 文字：深灰色文字"摄像头管理"
- 边框：无明显边框
- 整体：简洁、无装饰

**推测的 QSS 样式：**

```css
/* 基础/普通状态按钮 */
QPushButton {
    background-color: transparent;  /* 透明背景 */
    border: none;  /* 无边框 */
    border-radius: 4px;  /* 小圆角 */
    padding: 8px 16px;  /* 内边距 */
    color: #5f6368;  /* 深灰色文字 */
    text-align: left;  /* 文字左对齐 */
}
```

**关键属性说明：**
- **`transparent` 背景**：无背景色，保持简洁
- **无边框**：移除默认边框样式
- **小圆角**：提供轻微的圆角效果
- **左对齐**：图标和文字左对齐
