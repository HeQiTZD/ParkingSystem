无边框窗口设计方案

为 ConfigInitDialog 添加无边框样式和自定义标题栏

A

方案一：简洁标题栏

最小化 + 关闭按钮，无标题文字，纯图标按钮

优点

  • 简洁现代
  • 实现简单
  • 节省空间

缺点

  • 缺少窗口标题
  • 用户可能不知道窗口用途
B

方案二：带标题的标题栏

左侧显示"停车场数据初始化"，右侧最小化 + 关闭按钮

优点

  • 清晰的窗口标识
  • 专业感强
  • 用户友好

缺点

  • 需要更多空间
  • 实现稍复杂
C

方案三：可拖拽标题栏

带标题 + 最小化 + 关闭按钮，支持窗口拖拽移动

优点

  • 功能完整
  • 用户体验最佳
  • 符合桌面应用习惯

缺点

  • 实现复杂度最高
  • 需要处理鼠标事件

按钮悬浮样式预览

关闭按钮：红色悬浮 | 最小化按钮：浅灰色悬浮

关闭按钮样式
✕

鼠标悬浮变红

最小化按钮样式
─

鼠标悬浮变浅灰

实现要点

  • 使用 Qt::FramelessWindowHint 移除系统边框
  • 在 UI 文件中添加自定义标题栏布局
  • 通过 QSS 设置按钮悬浮样式
  • 可选：实现窗口拖拽功能
