# 数据库表格建立

## 概述

系统初始化时通过 `MySQLInit::initAll()` 创建以下4张表、2个触发器和1个定时任务。

数据库驱动：QMYSQL（MySQL）

---

## 1. User（用户表）

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | INT | PRIMARY KEY AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | UNIQUE NOT NULL | 用户名 |
| password | VARCHAR(64) | NOT NULL | 密码（SHA256） |
| telephone | VARCHAR(11) | DEFAULT '' | 手机号 |
| truename | VARCHAR(50) | DEFAULT '' | 真实姓名 |
| role | VARCHAR(10) | DEFAULT '' | 角色（admin/user） |
| create_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

索引：`idx_username(username)`

---

## 2. PARKING（停车场表）

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| P_id | INT | PRIMARY KEY AUTO_INCREMENT | 主键 |
| P_name | VARCHAR(100) | UNIQUE NOT NULL | 停车场名称 |
| P_now_count | INT | DEFAULT 0 | 现有车辆数 |
| P_reserve_count | INT | DEFAULT 0 | 预约车辆数 |
| P_all_count | INT | NOT NULL | 总车位数 |
| P_fee | DECIMAL(10,2) | NOT NULL | 每小时费用 |
| create_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

默认数据：`initParkingData("默认停车场", 100, 5.0)`

---

## 3. CAR（车辆记录表）

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | INT | PRIMARY KEY AUTO_INCREMENT | 主键 |
| license_plate | VARCHAR(20) | NOT NULL | 车牌号 |
| check_in_time | DATETIME | NOT NULL | 入库时间 |
| check_out_time | DATETIME | DEFAULT NULL | 出库时间（NULL=未出库） |
| fee | DECIMAL(10,2) | DEFAULT NULL | 停车费用 |
| create_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 记录创建时间 |

索引：
- `idx_license_plate(license_plate)`
- `idx_check_in_time(check_in_time)`

---

## 4. reservations（预约表）

| 字段 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | INT | PRIMARY KEY AUTO_INCREMENT | 主键 |
| license_plate | VARCHAR(20) | NOT NULL | 车牌号 |
| P_name | VARCHAR(100) | NOT NULL | 停车场名称 |
| create_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 预约时间 |

索引：`UNIQUE idx_license_plate(license_plate)`

---

## 触发器

### trg_reservation_insert
- **触发时机**：reservations 表 INSERT 之后
- **作用**：对应停车场 `P_reserve_count + 1`

### trg_reservation_delete
- **触发时机**：reservations 表 DELETE 之后
- **作用**：对应停车场 `P_reserve_count - 1`

---

## 定时任务

### clean_reservations
- **执行频率**：每1分钟
- **作用**：删除超过30分钟的过期预约
- **SQL**：`DELETE FROM reservations WHERE TIMESTAMPDIFF(MINUTE, create_at, NOW()) > 30`
