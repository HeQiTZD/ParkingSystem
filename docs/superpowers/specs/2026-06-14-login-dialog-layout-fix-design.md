# 登录界面布局修复设计

## 概述

修复 LoginDialog 中"用户登录"标题被遮挡和"忘记密码"标签显示不全的问题。

## 问题分析

### 问题 1：标题被遮挡

- **控件**：titleLabel
- **位置**：x=30, y=20
- **当前尺寸**：width=131, height=41
- **问题**：24pt 字体的"用户登录"需要约 160px 宽度，131px 不足以完整显示

### 问题 2：忘记密码显示不全

- **控件**：forgotPasswordLabel
- **位置**：x=470, y=310
- **当前尺寸**：width=51, height=21
- **问题**："忘记密码"4个字需要约 70px 宽度，51px 不足以完整显示

## 修复方案

仅调整控件尺寸和位置，保持绝对定位方式不变。

### 修改明细

| 控件 | 属性 | 当前值 | 修改为 |
|------|------|--------|--------|
| titleLabel | width | 131 | 200 |
| forgotPasswordLabel | x | 470 | 430 |
| forgotPasswordLabel | width | 51 | 100 |

## 验证标准

- [ ] "用户登录"标题完整显示，无遮挡
- [ ] "忘记密码"标签完整显示，右侧无截断
- [ ] 其他控件位置和功能不受影响
