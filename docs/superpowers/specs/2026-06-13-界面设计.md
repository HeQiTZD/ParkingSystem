# 车牌识别系统界面设计规范

## 1. 设计概述

### 1.1 设计风格
- **整体风格**：经典商务风格
- **设计原则**：专业可靠、简洁清晰、易于使用

### 1.2 设计规范
- **主色调**：天蓝色 (#3b82f6)
- **深色**：深蓝色 (#1e40af)
- **浅色**：浅蓝色 (#93c5fd)
- **背景色**：浅灰色 (#f8fafc)
- **字体**：微软雅黑
- **圆角**：4px
- **阴影**：中等阴影效果

## 3. 通用规范

### 3.1 颜色使用
- **主色调**：用于主要按钮、导航高亮、重要操作
- **深色**：用于侧边栏背景、品牌展示
- **浅色**：用于悬停状态、次要元素
- **成功色**：#10b981（绿色）- 用于成功状态
- **警告色**：#f59e0b（黄色）- 用于警告状态
- **错误色**：#ef4444（红色）- 用于错误状态、删除操作

### 3.2 字体规范
- **标题**：16-20px，加粗
- **正文**：14px，常规
- **辅助文字**：12px，灰色 (#64748b)

### 3.3 间距规范
- **页面内边距**：15px
- **卡片内边距**：15px
- **元素间距**：10-15px
- **表单元素间距**：12px

### 3.4 交互规范
- **按钮悬停**：颜色加深10%
- **输入框聚焦**：边框变为主色调
- **表格行悬停**：背景色变为浅灰色
- **禁用状态**：降低透明度，显示禁用样式
