# 车牌识别系统界面设计文档

## 1. 设计概述

### 1.1 设计目标
为车牌识别管理系统设计完整的用户界面，包括登录/注册、主功能界面、子页面和配置初始化窗口。

### 1.2 设计原则
- **浅色商务风格**：白色/浅灰背景，清晰简洁
- **蓝色系主色调**：专业稳重，适合管理系统
- **线条图标**：轻量简洁，现代感强
- **字体规范**：标题使用华文隶书，正文使用系统默认字体

### 1.3 技术方案
- **架构**：QWidget + QStackedWidget
- **主界面**：QMainWindow + 左侧导航栏 + 右侧 QStackedWidget
- **登录/注册/配置**：独立 QDialog，分栏式布局
- **页面切换**：无动画，直接切换

## 2. 登录窗口

### 2.1 布局设计
- **分栏式布局**：左侧品牌区 2/5 + 右侧表单区 3/5
- **固定窗口大小**：800×500，不可调整

### 2.2 左侧品牌区
- 蓝色渐变背景（#2563eb → #1e40af）
- 系统 Logo（🚗 图标）
- 系统名称："车牌识别系统"
- 副标题："智能停车场管理平台"

### 2.3 右侧表单区
- 标题："用户登录"
- 用户名输入框（标签 + 输入框）
- 密码输入框（标签 + 输入框，密码掩码显示）
- 按钮：登录（蓝色）、注册（白色描边）

### 2.4 交互逻辑
- 点击"登录"：验证用户名和密码，成功后跳转主功能界面
- 点击"注册"：跳转到注册窗口
- 登录失败：显示错误提示

- **分栏式布局**：左侧品牌区 2/5 + 右侧表单区 3/5

## 4. 主功能界面

### 4.1 整体布局
```
┌─────────────────────────────────────────┐
│ 左侧导航栏 │  顶部栏（时间显示）         │
│  80px      ├─────────────────────────────┤
│  深色背景   │  内容区域                   │
│            │  (QStackedWidget)           │
│  ┌──────┐  │                             │
│  │ 主页 │  │                             │
│  ├──────┤  │                             │
│  │摄像头│  │                             │
│  ├──────┤  │                             │
│  │车辆  │  │                             │
│  ├──────┤  │                             │
│  │管理  │  │                             │
│  └──────┘  │                             │
└─────────────────────────────────────────┘
```

### 4.2 左侧导航栏
- **宽度**：80px
- **背景色**：深色（#1e293b）
- **导航按钮**：60×60px，线条图标 + 文字
- **选中态**：蓝色背景（#2563eb），白色文字
- **未选中态**：透明背景，灰色文字（#94a3b8）
- **导航项**：主页、摄像头、车辆信息、管理

### 4.3 顶部栏
- **位置**：内容区域顶部，右对齐
- **内容**：时间显示（QLabel）
- **格式**：YYYY-MM-DD HH:mm:ss
- **样式**：深灰色（#334155），等宽字体（monospace），font-weight: 500
- **无背景框**，不遮挡内容区域

### 4.4 窗口特性
- **可调整大小**：支持窗口大小调整
- **内容区自适应**：导航栏固定 80px，内容区自动拉伸

## 5. 主页

## 6. 车辆信息页面

### 6.1 查询和排序区域
- **车牌号**：输入框
- **开始时间**：datetime-local 选择器
- **结束时间**：datetime-local 选择器
- **查询按钮**：蓝色
- **删除按钮**：红色
- **分隔线**：竖线分隔查询和排序
- **排序方式**：下拉选择（入库时间、出库时间、停车时长、停车费用）
- **排序方向**：下拉选择（正序 ↑、倒序 ↓）

**字体样式：** 标签 0.85rem / #1e293b / font-weight: 600（与表格一致）

### 6.2 数据表格
- **6列**：车牌号、入库时间、出库时间、停车时长、停车费用、状态
- **表头**：背景 #f1f5f9，文字 #1e293b，font-weight: 600
- **内容**：文字 #334155，font-weight: 500
- **字号**：0.85rem
- **状态标签**：
  - 在场：蓝色背景（#dbeafe），蓝色文字（#2563eb）
  - 已离场：绿色背景（#dcfce7），绿色文字（#16a34a）
- **选中行**：浅灰色高亮（#f0f9ff）
- **表格只读**，不可直接编辑

## 9. 配置初始化窗口

### 9.1 布局设计
- **分栏式布局**：左侧品牌区 2/5 + 右侧表单区 3/5
- **固定窗口大小**：不可调整
- **样式**：与登录/注册窗口一致

### 9.2 左侧品牌区
- 蓝色渐变背景
- 系统 Logo
- 系统名称："车牌识别系统"
- 副标题："停车场数据初始化"

### 9.3 右侧表单区
分为两个配置区域：

**数据库配置：**
- IP地址 + 端口号（并排）
- 数据库名 + 账户（并排）
- 密码（单独一行，密码掩码显示）

**停车场配置：**
- 停车场名（较大宽度）
- 价格（元/小时）+ 容量（车位数）（并排）

**按钮：** 提交（蓝色）、取消（灰色描边）

### 9.4 触发条件
- 首次启动时自动弹出（检测到没有配置文件）
- 数据库连接失败时提示进入配置

## 11. 页面切换逻辑

```

│  登录窗口 │ ───────────────→ │  主功能界面   │
└──────────┘                  └──────────────┘
      │                             │
      │ 点击注册                    │ 导航栏切换
      ↓                             ↓
┌──────────┐                  ┌──────────────┐
│  注册窗口 │                  │  子页面切换   │
└──────────┘                  │  (无动画)     │
      │                       └──────────────┘
      │ 点击返回
      ↓
┌──────────┐
│  登录窗口 │
└──────────┘

┌──────────────┐
│ 配置初始化窗口 │ ← 首次启动自动弹出
└──────────────┘
```
