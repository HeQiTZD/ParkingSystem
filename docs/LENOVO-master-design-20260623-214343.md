#### Phase 6: 收费规则与配置（2-3 天）

**收费规则：**
- 计费单位：半小时（0-30分钟按0.5小时，30-60分钟按1小时）
- 免费时段：15分钟（进场后15分钟内免费）
- 每日封顶：无
- 小时费率：可配置，存数据库，默认5元/小时

**计费公式：**
```
计费分钟 = max(0, 总停车分钟 - 15)
计费半小时数 = ceil(计费分钟 / 30)
费用 = 计费半小时数 × 小时费率 / 2
```

**数据库改动：**
- PARKING 表加 `P_free_minutes INT DEFAULT 15`
- CAR 表已有 `fee` 字段，不用改

**代码改动：**
1. Car::calculateFee 改签名：`double calculateFee(double hourlyRate, int freeMinutes = 15) const;`
2. DatabaseManager 加 getFeeRule()、updateFeeRule()、carCheckIn()、carCheckOut()
3. ConfigInitDialog 加"免费时长"输入框

### Error Handling Checklist

每个数据库操作：
```cpp
if (!db->someOperation()) {
    QMessageBox::critical(this, "错误", db->lastError());
    return;
}
```

摄像头操作：
```cpp
connect(cameraThread, &CameraThread::errorOccurred, this, [this](const QString &msg) {
    QMessageBox::warning(this, "摄像头错误", msg);
});
```

车牌识别：
```cpp
if (plateStr.isEmpty()) {
    QMessageBox::information(this, "提示", "未识别到车牌，请手动输入");
}
```
