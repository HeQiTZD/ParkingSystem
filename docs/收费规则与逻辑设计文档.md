# 停车场收费规则与逻辑设计文档

## 1. 概述

本文档定义了停车场管理系统的收费规则体系，旨在实现公平、合理、简洁的计费策略。

### 1.1 设计目标

- **公平性**：按实际使用时间计费，短时停车免费
- **简洁性**：统一费率，不区分车型，简单透明
- **可配置性**：关键参数可调整

### 1.2 基础费率

| 配置项 | 值 | 说明 |
|--------|-----|------|
| **基础费率** | 5.00 元/小时 | 统一费率 |

---

## 2. 核心收费规则

### 2.1 免费时段

- **30分钟内免费**：临时停靠、接送客人
- 超过30分钟开始计费

### 2.2 阶梯计费

采用"用得越久，单价越低"的阶梯定价策略：

| 停车时长 | 计费规则 | 实际单价 |
|----------|----------|----------|
| 0-30分钟 | **免费** | 0元 |
| 30分钟-2小时 | 基础费率 × 1.0 | 5元/小时 |
| 2-4小时 | 基础费率 × 0.9 | 4.5元/小时 |
| 4-8小时 | 基础费率 × 0.8 | 4元/小时 |
| 8-24小时 | 基础费率 × 0.7 | 3.5元/小时 |

### 2.3 每日封顶

- **封顶价：50元/天**
- 24小时为一个计费周期（从入场时间开始计算）
- 一个周期内费用不超过50元
- 超过24小时自动开始新周期

### 2.4 时段差异化

| 时段 | 时间范围 | 费率系数 |
|------|----------|----------|
| 日间 | 08:00-20:00 | 1.0（标准） |
| 夜间 | 20:00-次日08:00 | 0.6（6折） |

跨时段停车按实际时段分别计算。

## 3. 费用计算示例

### 示例1：短时停车（免费）
```
入场：09:00
出场：09:20
时长：20分钟 ≤ 30分钟
费用：0元
```

### 示例2：普通停车
```
入场：09:00
出场：12:30
时长：3.5小时
计算：
  - 前2小时：2 × 5 = 10元
  - 后1.5小时：1.5 × 5 × 0.9 = 6.75元
费用：16.75元
```

### 示例3：跨时段停车
```
入场：18:00
出场：23:00
时长：5小时
计算：
  - 日间(18:00-20:00)：2小时 × 5 × 1.0 = 10元
  - 夜间(20:00-23:00)：3小时 × 5 × 0.6 = 9元
费用：19元
```

### 示例4：长时间停车（触发封顶）
```
入场：09:00
出场：次日11:00
时长：26小时
计算：
  - 第一个24小时：封顶50元
  - 超出2小时：2 × 5 × 1.0 = 10元
费用：60元
```

## 4. 计费算法

### 4.1 核心计费函数

```cpp
/**
 * @brief 计算停车费用
 * @param checkInTime 入场时间
 * @param checkOutTime 出场时间
 * @param baseRate 基础费率（默认5.0元/小时）
 * @return 应收费用（元）
 */
double calculateFee(
    const QDateTime& checkInTime,
    const QDateTime& checkOutTime,
    double baseRate = 5.0)
{
    // 1. 计算停车时长（分钟）
    int totalMinutes = checkInTime.secsTo(checkOutTime) / 60;
    
    // 2. 免费时段判断（30分钟）
    if (totalMinutes <= 30) {
        return 0.0;
    }
    
    // 3. 按时段分割计算（日间/夜间）
    double totalFee = 0.0;
    auto segments = splitByTimePeriod(checkInTime, checkOutTime);
    
    for (const auto& segment : segments) {
        double segmentHours = segment.minutes / 60.0;
        double segmentFee = applyTieredRate(segmentHours, baseRate);
        segmentFee *= segment.periodMultiplier; // 应用时段系数
        totalFee += segmentFee;
    }
    
    // 4. 应用每日封顶（50元/天）
    int days = (totalMinutes + 1439) / 1440; // 向上取整天数
    double maxFee = 50.0 * days;
    totalFee = std::min(totalFee, maxFee);
    
    // 5. 四舍五入到分
    return std::round(totalFee * 100) / 100;
}
```

### 4.2 阶梯费率计算

```cpp
/**
 * @brief 应用阶梯费率
 * @param hours 停车小时数
 * @param baseRate 基础费率
 * @return 阶梯计算后的费用
 */
double applyTieredRate(double hours, double baseRate)
{
    double fee = 0.0;
    double remaining = hours;
    
    // 阶梯规则：[时长上限, 费率倍数]
    struct Tier { double maxHours; double multiplier; };
    std::vector<Tier> tiers = {
        {2.0, 1.0},   // 0-2小时：100%
        {2.0, 0.9},   // 2-4小时：90%
        {4.0, 0.8},   // 4-8小时：80%
        {16.0, 0.7},  // 8-24小时：70%
    };
    
    for (const auto& tier : tiers) {
        if (remaining <= 0) break;
        double hoursInTier = std::min(remaining, tier.maxHours);
        fee += hoursInTier * baseRate * tier.multiplier;
        remaining -= hoursInTier;
    }
    
    // 超过24小时按基础费率
    if (remaining > 0) {
        fee += remaining * baseRate;
    }
    
    return fee;
}
```

## 5. 数据库设计

### 5.1 车辆记录表（CAR）

```sql
CREATE TABLE IF NOT EXISTS CAR (
    id INT PRIMARY KEY AUTO_INCREMENT,
    license_plate VARCHAR(20) NOT NULL COMMENT '车牌号',
    check_in_time DATETIME NOT NULL COMMENT '入库时间',
    check_out_time DATETIME DEFAULT NULL COMMENT '出库时间',
    parking_minutes INT DEFAULT 0 COMMENT '停车时长(分钟)',
    final_fee DECIMAL(10,2) DEFAULT 0.00 COMMENT '最终费用',
    payment_status TINYINT DEFAULT 0 COMMENT '支付状态(0未付/1已付)',
    payment_time DATETIME COMMENT '支付时间',
    location VARCHAR(50) COMMENT '停车位置',
    create_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_license_plate (license_plate),
    INDEX idx_check_in_time (check_in_time)
) COMMENT='车辆停车记录表';
```

### 5.2 停车场配置表（PARKING）

```sql
CREATE TABLE IF NOT EXISTS PARKING (
    P_id INT PRIMARY KEY AUTO_INCREMENT,
    P_name VARCHAR(100) UNIQUE NOT NULL COMMENT '停车场名称',
    P_now_count INT DEFAULT 0 COMMENT '现有车辆数',
    P_all_count INT NOT NULL COMMENT '总车位数',
    P_fee DECIMAL(10,2) NOT NULL DEFAULT 5.00 COMMENT '每小时费用',
    P_free_minutes INT DEFAULT 30 COMMENT '免费时长(分钟)',
    P_daily_cap DECIMAL(10,2) DEFAULT 50.00 COMMENT '每日封顶价',
    P_day_start INT DEFAULT 8 COMMENT '日间开始时间(小时)',
    P_day_end INT DEFAULT 20 COMMENT '日间结束时间(小时)',
    P_night_discount DECIMAL(3,2) DEFAULT 0.60 COMMENT '夜间折扣系数',
    create_at DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT='停车场配置表';
```

### 5.3 系统配置表（SYSTEM_CONFIG）

```sql
CREATE TABLE IF NOT EXISTS SYSTEM_CONFIG (
```

## 6. 业务流程

### 6.1 车辆入场

```
识别车牌 → 查询记录 → 创建停车记录(check_in_time) → 更新车位统计 → 入场完成
```

### 6.2 车辆出场

```
识别车牌 → 查询在场记录 → 计算停车时长
                              ↓
                    ┌─────────┴─────────┐
                    │                   │
               ≤30分钟              >30分钟
                    │                   │
                 免费放行          计算费用
                                      ↓
                              ┌───────┴───────┐
                              │               │
                          阶梯计费        时段差异
                              │               │
                              └───────┬───────┘
                                      ↓
                                判断是否封顶
                                      ↓
                                 显示费用
                                      ↓
                                   支付
                                      ↓
                            更新记录(check_out_time, final_fee)
                                      ↓
                               更新车位统计
                                      ↓
                                   放行
```
