# VehicleEntryExitWidget 自定义控件提升方案

> 设计文档版本:v1.0
> 控件目标:将 `mainwindow.ui` 中的 **占位 QWidget** `VehicleEntryExitWidget` **提升(promote)为真正的自定义控件**,展示车牌 + 时间 + 入库/出库状态,采用 Qt C++ Widgets。
> 范围:仪表盘首页右栏「最新记录」卡片内,仅此一处复用(放 `UI/MainWindow/`）。

## 2. 总体架构

采用 **Qt Promote(提升) + 自包含 Layout** 方式,分三层解耦:

```
┌─────────────────────────────────────────────────────────────┐
│   MainWindow (宿主,本方案零改动)                             │
│                                                             │
│   ┌─ ui_mainwindow.h 自动生成占位 ────────────────────────┐ │
│   │   QWidget* VehicleEntryExitWidget  (占位 QWidget)     │ │
│   └──────────────────────────────────────────────────────┘ │
│                       │ 提升后 Designer 知道此类             │
│                       ▼                                      │
│   ┌─新的自定义类──────────────────────────────────────────┐│
│   │  VehicleEntryExitWidget : public QWidget              ││
│   │  (提升用的类,内嵌行控件 + 垂直列表 + 分隔线)           ││
│   │                                                       ││
│   │   ┌─ VehicleEntryItem (内部类,单条记录行控件) ────┐  ││
│   │   │  [车牌号               ]  [状态标签(蓝/灰)]   │  ││
│   │   │  [时间 HH:mm:ss        ]                      │  ││
│   │   └──────────────────────────────────────────────┘  ││
│   │   ├─ 分隔线(HLine)                                  ││
│   │   │   #2 ...                                         ││
│   └──────────────────────────────────────────────────────┘└
└─────────────────────────────────────────────────────────────┘
                         ▲
                         │ 数据驱动(setEntries / addEntry)
                         │
                  DatabaseManager(信号,预留,当前自包含示例数据)
```

**为什么选 promote?**
- Qt 原生做法,Designer 可视化。
- 与 `CircleProgress` 挂 `parkingChartWidget` 是**镜像操作**,架构一致。
- 提升后 `ui->VehicleEntryExitWidget` 类型自动变成自定义类 → **后续调用 `setEntries()` 等 API 无需额外查找**。

## 3. 数据结构规格

**位置**:`src/widgets/vehicleentry.h`

```cpp
#ifndef VEHICLEENTRY_H
#define VEHICLEENTRY_H

#include <QString>
#include <QDateTime>

// 入库/出库状态。蓝色=入库, 浅灰=出库(对齐需求)。
enum class VehicleEntryStatus { In, Out };

// 纯数据载体(POD),跨 DB/UI 层共用,不依赖 QWidget。
// 车牌建议在 DB 层拼好 "苏A·12345" 格式,UI 层只负责显示。
struct VehicleEntry {
    QString plate;           // 已格式化车牌,如 "苏A·12345"(U+00B7 中位点)
    QDateTime timestamp;     // 事件时间
    VehicleEntryStatus status;

    // 便捷访问器
    bool     isEntry()     const { return status == VehicleEntryStatus::In; }
    QString  timeStr()     const { return timestamp.toString("HH:mm:ss"); }
    QString  displayText() const {
        return QStringLiteral("%1 · %2 · %3")
            .arg(plate, timeStr(), isEntry() ? "入库" : "出库");
    }
};

#endif
```

> ⚠️ 格式化点建议在**数据层**(数据库读取 / 识别结果转存)就拼好 `苏A·12345` 格式字符串,控件只负责**显示**。这样换皮或换语言时,显示层零改动。
> 中间点用 **U+00B7 中位点**(·),不用英文句点(.)。

## 6. 视觉规格(精确到 px)

### 6.1 布局(单条记录,64px 高)

```
←──── 整行高度 64px ────→
┌─────────────────────────────────────────────┐
│ 苏A·12345                      ┌────────┐   │  ← 车牌 17sp 中粗 #1F2937
│ 10:45:22                       │  入库  │   │  ← 时间 12sp 正常 #6B7280
│                                └────────┘   │    右上:状态标签
└─────────────────────────────────────────────┘
←12px→            ←stretch→   ←8px→  ←56px→
```

### 6.2 元素样式表

| 元素 | 字号 | 字重 | 颜色 | 其它 |
|------|------|------|------|------|
| 车牌号 | 17sp(≈22px) | 600(Medium) | `#1F2937` | 左对齐 |
| 时间 | 12sp(≈16px) | 400(Normal) | `#6B7280` | 左对齐,跟随车牌列 |
| 行高 | **64px 固定** | — | — | 保证列表整齐 |
| 行内水平 padding | 左 12px / 右 12px | — | — | 上下 4px |
| 中间 spacing | 8px | — | — | 车牌区与状态标签之间 |

### 6.3 状态标签

| 状态 | 背景色 | 文字色 | QSS state 属性 |
|------|--------|--------|----------------|
| 入库 | `#2563EB`(蓝) | `#FFFFFF`(白) | `"in"` |
| 出库 | `#F3F4F6`(浅灰) | `#4B5563`(深灰) | `"out"` |

圆角:**12px 胶囊形**(若要小方角改 4px)。
最小宽度:**56px**,高 24px,文字居中。

> 💡 **QSS 伪状态动态刷新关键**:切换状态时调用 `style()->unpolish(widget); style()->polish(widget);`,否则 `[state="in"]` 伪状态不会动态更新。此模式已在 `ToastWidget::buildUi` 中使用。

### 6.4 分隔线

```css
QFrame[frameShape="HLine"] {
    color: #E5E7EB;          /* 浅灰 gray-200 */
    max-height: 1px;
}
```

使用 `QFrame(nullptr, Qt::HLine)`,设 `max-height:1px`,避免高分屏下发虚。**最后一行不加分隔线**,避免和卡片下边缘重叠。

## 8. 可访问性 & 对比度验收

| 项 | 前景 / 背景 | 对比度 | WCAG AA | 备注 |
|----|-------------|--------|---------|------|
| 车牌 | `#1F2937` on `#FFFFFF` | ≈ 10.4:1 | ✅ | — |
| 时间 | `#6B7280` on `#FFFFFF` | ≈ 4.7:1 | ✅ | **已修正**,原需求 `#9CA3AF` 仅 2.6:1 不达标 |
| 入库标签 | `#FFFFFF` on `#2563EB` | ≈ 4.6:1 | ✅ 勉强 | — |

**发现的对比度问题 & 修正**:

### 13.4 F1 — `src/widgets/vehicleentry.h`(数据层,对齐原§3 保留)

> 创建方式:**新建文件** `src/widgets/vehicleentry.h`,复制下面**整个代码块**,文件首部对齐项目已有的头文件样式(如 `src/utils/toastwidget.h` 的 `#ifndef` 三件套)。

```cpp
#ifndef VEHICLEENTRY_H
#define VEHICLEENTRY_H

// ****************************************************************************
//  VehicleEntry — 车辆进出记录纯数据载体(POD)
//  位置        :src/widgets/vehicleentry.h
//  职责        :跨 DB / UI 层共用数据结构,不依赖 QWidget
//  对齐文档    :v1.1 §13 取代原 §5 子控件方案 / 保留原 §3 数据结构
//  使用方      :
//    – DatabaseManager 读写时构造(建议在 DB 层就拼好 "苏A·12345" 格式)
//    – VehicleEntryExitWidget 内核遍历 m_entries 绘制
//  设计决策    :车牌中间的点用 U+00B7 中位点(·),不用英文句点
// ****************************************************************************

#include <QString>
#include <QDateTime>

// 入库/出库状态。蓝色=入库 (#2563EB 白字),浅灰=出库 (#F3F4F6 深灰字)
enum class VehicleEntryStatus { In, Out };

// POD 结构 — 没有任何 QWidget 依赖,可在单元测试/DBAccess 层直接构造
struct VehicleEntry {
    QString plate;           // 已格式化车牌,如 "苏A·12345" ( U+00B7 中位点 )
    QDateTime timestamp;     // 事件时间
    VehicleEntryStatus status;

    // 便捷访问器(与 VehicleEntryExitWidget::paintEvent 配合,省下重复分支)
    bool isEntry() const { return status == VehicleEntryStatus::In; }
    // 显示格式对齐正文档 §6.5 示例数据的时间格式 "10:45:22"
    QString timeStr() const { return timestamp.toString("HH:mm:ss"); }
    // QAccessibleText 朗读用:"苏A12345, 10点45分, 入库"
    QString displayText() const {
        return QStringLiteral("%1, %2, %3")
            .arg(plate, timeStr(), isEntry() ? QStringLiteral("入库") : QStringLiteral("出库"));
    }
};

#endif // VEHICLEENTRY_H
```

> ⚠️ **关键点**:一个字都不用改,剪贴即可。后续 DatabaseManager 增加查询时,结果集直接装 `QList<VehicleEntry>`。

#include <QPainter>
#include <QWheelEvent>
#include <QFont>
#include <QStyleOption>
#include <QtMath>

// 颜色常量对齐正文档 §8 验收清单(背景白 #FFFFFF,字体/标签色 WCAG AA)
// 这些颜色硬编码 — 自绘不设 objectName,QSS 选色器失效(见 v1.1 §13.13)

// ===== 公共 API =====
void VehicleEntryExitWidget::setEntries(const QList<VehicleEntry> &entries)
{
    m_entries = entries;
    if (m_entries.size() > m_maxCount)
        m_entries = m_entries.mid(0, m_maxCount);
    m_scrollY = 0;   // 重置滚动(避免视口留在底部看到空白)
    update();
}

void VehicleEntryExitWidget::prependEntry(const VehicleEntry &entry)
{
    m_entries.prepend(entry);
    while (m_entries.size() > m_maxCount)
        m_entries.removeLast();
    clampScrollY();
    update();
}

void VehicleEntryExitWidget::clear()
{
    m_entries.clear();
    m_scrollY = 0;
    update();
}

void VehicleEntryExitWidget::setMaxCount(int n)
{
    m_maxCount = qMax(1, n);
    clampScrollY();
    update();
}

// ===== 私有工具 — 滚动范围计算 =====
int VehicleEntryExitWidget::contentHeight() const
{
    if (m_entries.isEmpty()) return 0;
    return m_entries.size() * kRowHeight
         + (m_entries.size() - 1) * kSeparator;
}

int VehicleEntryExitWidget::maxScrollY() const
{
    return qMax(0, contentHeight() - height());
}

void VehicleEntryExitWidget::clampScrollY()
{
    m_scrollY = qBound(0, m_scrollY, maxScrollY());
}

// ===== wheelEvent — 滚轮自接管滚动 =====
// 不创建任何滚动条控件 → 仅调整 m_scrollY(像素偏移),paintEvent 内部用偏移绘制
void VehicleEntryExitWidget::wheelEvent(QWheelEvent *event)
{
    if (m_entries.isEmpty() || maxScrollY() == 0) {
        event->ignore();
        return;
    }
    // angleDelta().y():上滚为正,下滚为负;乘以系数控制滚动速度
    int delta = event->angleDelta().y();
    m_scrollY = qBound(0, m_scrollY - delta, maxScrollY());
    update();
    event->accept();
}

// ===== paintEvent — 整个控件的所有视觉内容 =====
// 顺序:剪贴板背景 → 每条记录(车牌+时间+标签)+ 分隔线 → 溢出提示渐变
void VehicleEntryExitWidget::paintEvent(QPaintEvent *event)
{
    Q_UNUSED(event);
    QPainter p(this);
    p.setRenderHint(QPainter::Antialiasing, true);
    p.setRenderHint(QPainter::TextAntialiasing, true);

    // 1. 画卡片圆角背景
    p.setPen(Qt::NoPen);
    p.setBrush(kColorCardBg);
    p.drawRoundedRect(rect(), kCardRadius, kCardRadius);

    if (m_entries.isEmpty()) {
        // 扩展点:可在这里画空态占位(暂无记录);简单起见先留白
        return;
    }

    // 2. 裁剪到圆角矩形内(避免绘制溢出)
    QPainterPath clipPath;
    clipPath.addRoundedRect(rect(), kCardRadius, kCardRadius);
    p.setClipPath(clipPath);

    // 3. 遍历每条记录绘制
    const int rowStride = kRowHeight + kSeparator;
    for (int i = 0; i < m_entries.size(); ++i) {
        const VehicleEntry &e = m_entries[i];
        int y = i * rowStride - m_scrollY;

        // 剪掉视口外的行(显著减少绘制开销,10 条数据下不明显,但习惯保留)
        if (y + kRowHeight < 0 || y > height()) continue;

        // 3.1 车牌 — 17sp Medium,深灰
        QFont f = font();
        f.setPixelSize(kFontSizePlate);
        f.setWeight(QFont::Medium);
        p.setFont(f);
        p.setPen(kColorPlateText);
        p.drawText(kTextPadding, y + 24, e.plate);   // 基线 y+24 让文字在行内视觉居中偏上

        // 3.2 时间 — 12sp Normal,中灰(对齐 §8 修正)
        f.setPixelSize(kFontSizeTime);
        f.setWeight(QFont::Normal);
        p.setFont(f);
        p.setPen(kColorTimeText);
        p.drawText(kTextPadding, y + 48, e.timeStr());

        // 3.3 状态标签 — 圆角矩形 + 居中字
        QRect badge(width() - kTextPadding - kBadgeWidth, y + (kRowHeight - kBadgeHeight) / 2,
                   kBadgeWidth, kBadgeHeight);
        p.setPen(Qt::NoPen);
        p.setBrush(e.isEntry() ? kColorInBadgeBg : kColorOutBadgeBg);
        p.drawRoundedRect(badge, kBadgeRadius, kBadgeRadius);
        p.setFont(QFont(font().family(), kFontSizeBadge, QFont::Medium));
        p.setPen(e.isEntry() ? kColorInBadgeText : kColorOutBadgeText);
        p.drawText(badge, Qt::AlignCenter, e.isEntry() ? QStringLiteral("入库") : QStringLiteral("出库"));

        // 3.4 分隔线 — 最后一行不画(避免贴在卡片下边缘)
        if (i + 1 < m_entries.size()) {
            p.setPen(QPen(kColorSeparator, 1));
            p.drawLine(kTextPadding, y + kRowHeight,
                       width() - kTextPadding, y + kRowHeight);
        }
    }

    // 4. 顶部/底部渐变提示溢出(因为没有滚动条,这是用户唯一的可滚动视觉线索)
    drawEdgeFade(p);
}

// 顶部/底部渐变 — 当 m_scrollY>0 时顶部渐变;当 m_scrollY<maxScrollY 时底部渐变
void VehicleEntryExitWidget::drawEdgeFade(QPainter &p)
{
    const int fadeH = 12;
    if (m_scrollY > 0) {
        QLinearGradient topFade(0, 0, 0, fadeH);
        topFade.setColorAt(0.0, kColorFadeEdge);
        topFade.setColorAt(1.0, Qt::transparent);
        p.fillRect(0, 0, width(), fadeH, topFade);
    }
    int maxSY = maxScrollY();
    if (m_scrollY < maxSY) {
        QLinearGradient bottomFade(0, height() - fadeH, 0, height());
        bottomFade.setColorAt(0.0, Qt::transparent);
        bottomFade.setColorAt(1.0, kColorFadeEdge);
        p.fillRect(0, height() - fadeH, width(), fadeH, bottomFade);
    }
}

### 13.11 关键反模式提醒(v1.1 专用)

| 反模式 | 为什么不行 | v1.1 怎么做 |
|--------|------------|-------------|
| ❌ 改用 `QScrollArea` 替代 `wheelEvent` 自接管 | 滚动条不可见约束不满足(QScrollArea 自带可视滚动条控件) | ✅ `wheelEvent` + `m_scrollY` 偏移绘制 |
| ❌ 保留 QLabel 子控件组合 | 违反"自定义绘制"硬约束(正文档末用户指示) | ✅ 全部 painter 自绘 |
| ❌ 让控件 objectName 依赖 QSS 设色 | 自绘设色不走 QSS;painter 设 QColor 硬编码即可 | ✅ constexpr QColor 常量 |
| ❌ paintEvent 里写 `exec()` / 模态对话框 | 绘制函数不能阻塞 → 事件循环卡死 | ✅ paintEvent 只发 signal,MainWindow 接 |
| ❌ 多次 setEntries 调用导致内存泄漏 | paintEvent 不开 Item 子控件,不存在 Item 泄漏 | ✅ m_entries 是 QList\<POD\>,无 new 操作 |
| ❌ 在 wheelEvent 里 `update()` 不限制 m_scrollY 范围 | 滚动过头出现空白 | ✅ `qBound(0, m_scrollY, maxScrollY())` 夹住 |

### 13.12 触摸 + Mac 触控板 + Windows 触控板 — 设备兼容性说明

> 这部分是 v1.1 **验收前必知**:自接管滚动对不同输入设备的响应差异。

| 设备 | Qt 映射 | v1.1 体验 |
|------|---------|-----------|
| **小鼠滚轮** | `QWheelEvent::angleDelta().y()` 为 ±120°(1 条约 72px 偏移) | ✅ 流畅 |
| **Mac 触控板双指拖动** | Qt 把惯性滚动转成连续小 delta 的 `QWheelEvent`(macOS 原生惯性) | ✅ 平滑带惯性 |
| **Windows 触控板** | 和 Mac 类似,但部分驱动步长略粗 | ✅ 可用,但惯性略硬 |
| **触屏手指拖动(移动端)** | 不产生 `QWheelEvent`;需要 `QGestureEvent` / `QTouchEvent` 实现 | ⚠️ **本需求是桌面程序,可暂不做** |

> ⚠️ 项目是 Qt Widgets 桌面应用(程序 `main.cpp` 走 `QApplication::exec()`),主流输入是鼠标滚轮;后续若要做触屏扩展,在 `VehicleEntryExitWidget` 加 `QGestureEvent` 即可,现阶段不做避免过度设计。
