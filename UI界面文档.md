# UI 界面文档

## 项目概述
基于 Qt 5.x 的车牌识别系统界面设计文档，包含登录、注册、主功能、配置初始化四个界面。

---

## 界面总览

| 界面文件 | 类名 | 尺寸 | 用途 |
|---------|------|------|------|
| `mainwindow.ui` | MainWindow | 781×543 | 登录窗口 |
| `signup.ui` | Signup | 780×540 | 注册窗口 |
| `car.ui` | Car | 1482×715 | 主功能界面 |
| `initfile.ui` | initFile | 629×591 | 配置初始化窗口 |

## 5. 设计规范

### 5.1 颜色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 背景色 | `rgb(255, 255, 255)` | 白色 |
| 按钮主色 | `rgb(0, 170, 127)` | 绿色 |
| 按钮渐变 | `rgba(64, 145, 232/252, 255)` → `rgba(255, 255, 255, 255)` | 蓝色渐变 |
| 输入框背景 | `rgb(247, 247, 247)` | 浅灰色 |
| 取消按钮 | `rgb(255, 85, 0)` | 橙色 |
| 文字颜色 | `rgb(255, 255, 255)` | 白色（按钮上） |

### 5.2 字体规范

| 字体 | 大小 | 用途 |
|------|------|------|
| 华文隶书 | 20pt | 大标题（账户、密码标签） |
| 华文隶书 | 15pt | 中标题（注册表单标签） |
| 华文隶书 | 13pt | 小标题（识别车牌号） |
| 华文隶书 | 10pt | 输入框文字 |
| 字心坊猫的天空之城 | 15pt/20pt | 注册/配置页面标题 |
| Bahnschrift Light | 9pt | 按钮文字 |

### 5.3 圆角规范

- **输入框**: `border-radius: 7px`
- **按钮**: `border-radius: 7px`
- **边框**: `border: 1px groove gray`

### 5.4 间距规范

- **输入框高度**: 45-61px
- **按钮高度**: 45-61px
- **垂直间距**: 使用 QSpacer (40px)
- **水平间距**: 使用 QSpacer (20-40px)

## 7. 开发备注

1. **界面固定大小**: 登录、注册、配置界面均为固定大小，不可调整
2. **主界面**: car.ui 为最复杂的界面，使用 QStackedWidget 实现多页面切换
3. **样式统一**: 所有界面使用相同的颜色、字体、圆角规范
4. **中文支持**: 使用华文隶书、字心坊猫的天空之城等中文字体
5. **输入验证**: 密码输入框使用 QLineEdit::Password 模式
6. **表格只读**: car.ui 中的表格设置为 NoEditTriggers
