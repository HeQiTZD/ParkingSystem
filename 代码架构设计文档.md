## 1. 项目概述

### 1.1 系统简介

基于 Qt C++ 开发的停车场管理系统，集成 EasyPR 车牌识别库，实现车辆入库/出库、车牌自动识别、收费管理、用户管理等功能。

### 1.2 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Qt | 6.x | UI 框架、信号槽、数据库访问 |
| C++ | 11 | 核心语言 |
| MySQL | 5.7+ | 数据存储 |
| OpenCV | 3.4.8 | 图像处理、摄像头捕获 |
| EasyPR | - | 车牌识别 |
| Qt Charts | - | 数据可视化 |
| QSS | - | UI 样式 |

### 1.3 构建系统

- **构建工具:** qmake (ParkingSystem.pro)
- **编译器:** MinGW
- **输出:** ParkingSystem.exe

**费用计算算法:**

```cpp
double Car::calculateFee(double hourlyRate, int freeMinutes) const {
    if (!isParked()) return 0.0;

    // 计算总停车分钟数
    qint64 totalMinutes = m_checkInTime.secsTo(
        m_checkOutTime.isValid() ? m_checkOutTime : QDateTime::currentDateTime()
    ) / 60;

    // 减去免费时段
    int billableMinutes = qMax(0LL, totalMinutes - freeMinutes);

    // 按半小时为单位向上取整
    int halfHourUnits = qCeil(billableMinutes / 30.0);

    // 计算费用
    return halfHourUnits * hourlyRate / 2.0;
}
```

**车牌验证规则:**
- 长度：7 或 8 位
- 第一位：省份简称（京津沪渝冀豫云辽黑湘皖鲁新苏浙赣鄂桂甘晋蒙陕吉闽贵粤川青藏琼宁）
- 第二位：字母 A-Z
- 后续：字母和数字组合

**数据库表结构:**

```sql
-- 用户表
CREATE TABLE IF NOT EXISTS User(
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL COMMENT '用户名',
    password VARCHAR(64) NOT NULL COMMENT '密码(SHA256)',
    telephone VARCHAR(11) DEFAULT '' COMMENT '手机号',
    truename VARCHAR(50) DEFAULT '' COMMENT '真实姓名',
    role VARCHAR(10) DEFAULT '' COMMENT '角色(admin/user)',

    INDEX idx_username(username)

CREATE TABLE IF NOT EXISTS CAR(
```

**线程安全:**
- `cv::Mat` 通过信号槽传递，Qt 自动处理跨线程安全
- 使用 `Qt::QueuedConnection` 确保槽函数在主线程执行
- `m_running` 标志位用于优雅停止线程
- `m_paused` + `QWaitCondition m_pauseCond` 用于暂停/恢复机制
- `QMutex m_mutex` 保护 `m_latestFrame` 的线程安全访问

**暂停/恢复机制:**
```cpp
// 暂停：设置标志，下次循环进入等待
void CameraThread::pause() {
    QMutexLocker locker(&m_mutex);
    m_paused = true;
}

// 恢复：唤醒等待的线程
void CameraThread::resume() {
    QMutexLocker locker(&m_mutex);
    m_paused = false;
    m_pauseCond.wakeOne();
}

// run() 循环中检查暂停状态
void CameraThread::run() {
    while (m_running) {
        // 检查暂停
        {
            QMutexLocker locker(&m_mutex);
            while (m_paused) {
                m_pauseCond.wait(&m_mutex);
            }
        }
        // 捕获帧...
    }
}
```

```cpp
query.exec();

// 错误：字符串拼接（禁止）
QString sql = QString("SELECT * FROM User WHERE username='%1'").arg(username);
```

### 12.3 线程安全注意事项

3. **共享数据** — 使用 QMutex 或 QReadWriteLock 保护
4. **线程停止** — 使用标志位 + wait() 优雅停止


### 13.2 单例生命周期

PlateRecognize 使用单例模式，程序退出时由操作系统回收。

```cpp
// 单例获取
PlateRecognize* PlateRecognize::instance() {

```
